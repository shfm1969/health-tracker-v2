# 個人健康數據追蹤系統 (Health Tracker) v2.0 操作手冊

本系統是一個基於 Python 與 SQLite 的圖形化介面 (GUI) 程式，旨在幫助使用者管理多人的血壓、脈搏及體重等健康數據。

## 1. 快速啟動 (Windows)

如果您已安裝 `uv`（如本專案環境），可以使用以下指令執行：

```bash
uv run health_tracker_gui.py
```

或者，如果您已啟用虛擬環境：

```bash
python health_tracker_gui.py
```

## 2. 第一次使用流程

1. **啟動程式**：執行後您會看到主視窗，頂端顯示 `[ 當前使用者: 未登入 ]`。
2. **新增使用者**：
   - 點擊「選擇使用者」按鈕。
   - 在彈出視窗中點擊「新增使用者」按鈕。
   - 依序輸入姓名、年齡、性別。
   - 完成後，系統會自動將您切換為當前使用者。
3. **登錄健康紀錄**：
   - 返回主視窗後，點擊「登入紀錄」按鈕。
   - 彈出視窗會顯示當前時間和上一次體重（若有）。
   - 依序輸入收縮壓、舒張壓、脈搏、體重及測量部位。
   - *提示：第二次登錄時，體重會預設帶入上一次的數值，可直接儲存。*
4. **查看歷史紀錄**：
   - 點擊「查看歷史資料」按鈕，即可看到按時間倒序排列的歷史數據。

## 3. 功能說明

- **狀態意識**：程式頂端始終顯示目前是誰在操作，確保數據不會錄錯人。
- **自動帶入**：登錄紀錄時，系統會自動獲取當前時間與該使用者的上一次體重作為預設值。
- **資料持久化**：所有數據儲存在 `health_data.db` 檔案中，關閉程式後數據不會消失。

## 4. 系統需求
- Python 3.13+ (本專案環境建議)
- SQLite3 (Python 內建)
- tkinter (Python 內建)
